# University/Organization Portal System

A comprehensive web application for managing user authentication, profiles, departments, and leave management for educational institutions and organizations.

## Features

### User Management
- **Sign Up**: Email, Name, Age, Birthday, Gender, Location, Password
- **Account Setup**: Focal person, Employee/Student type, ID number, Department, Campus, Code number
- **Login**: Username/Email and Password
- **Forgot Password**: Email-based password recovery

### Department Management
- Location-based department organization
- Campus-wise filtering
- Detailed department information
- Interactive campus map

### Leave Management
- Request leave with type selection
- Date range selection
- Leave history tracking
- Interactive calendar view
- Leave policy information

### Profile Management
- Multi-step profile setup
- Progress tracking
- Profile completion validation
- User information review
